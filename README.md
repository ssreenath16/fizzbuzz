# Fizz Buzz Application using Springboot in java
This Program generally follows this rules ggiven below
#####For any integer value:
- If the integer is divisible by 3: return “fizz”
- If the integer is divisible by 5: return “buzz”
- If the integer is divisible by 3 and 5: return “fizzbuzz”
- Otherwise return a string representation of the integer

##Docker File
It uses Open JDK 8 to run the application.

######Runs in Port : 8080
##API's
- /getFizzBuzzBetweenNumber
> API to see the FizzBuzz between two Numbers
- /getFizzBuzzList
> API to see the FizzBuzz between 1 and input number
